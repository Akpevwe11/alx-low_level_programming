## 0. Linear search 

### Task 0:

Write a function that searches for a value in an array of integers using the Linear search algorithm

 **Requirements:**

   [x] Prototype : int linear_search(int *array, size_t size, int value);
   [x] Where array is a pointer to the first element of the array to search in
   [x] size is the number of elements in array
   [x] And value is the value to search for
   [x] Your function must return the first index where value is located
   [x] If value is not present in array or if array is NULL, your function must return -1
   [x]  Every time you compare a value in the array to the value you are searching, you have to print this value (see example below)

Example:

```bash
wilfried@0x1E-search_algorithms$ cat 0-main.c 
#include <stdio.h>
#include <stdlib.h>
#include "search_algos.h"

/**
 * main - Entry point
 *
 * Return: Always EXIT_SUCCESS
 */
int main(void)
{
    int array[] = {
        10, 1, 42, 3, 4, 42, 6, 7, -1, 9
    };
    size_t size = sizeof(array) / sizeof(array[0]);

    printf("Found %d at index: %d\n\n", 3, linear_search(array, size, 3));
    printf("Found %d at index: %d\n\n", 42, linear_search(array, size, 42));
    printf("Found %d at index: %d\n", 999, linear_search(array, size, 999));
    return (EXIT_SUCCESS);
}
```
